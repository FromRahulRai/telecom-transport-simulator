# Telecom Transport Network Simulator

## Objective
Simulate a real-world telecom transport network including:
- BTS to Aggregation to Core topology
- OSPF routing
- BGP peering simulation
- VLAN segmentation
- ACL security implementation
- Python-based configuration validation

## Project Structure
src/        -> Automation scripts
configs/    -> Router configuration templates
topology/   -> Network design diagrams
docs/       -> Technical documentation
